About NeoWeather

NeoWeather is a futuristic Flask-based web application that provides real-time weather data for multiple cities using the OpenWeatherMap API.

Tech Stack:

  • Python & Flask
  • OpenWeatherMap API
  • TailwindCSS & Alpine.js

Features:

  • Multi-city weather tracking
  • Responsive, futuristic design
  • Real-time data updates
